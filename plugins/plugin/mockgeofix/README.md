# MockGeoFix
Pushes Coordinates to a TCP Server running on Android

Required Flags:
* -plugin "mockgeofix"
* -plugin-addr "IP:5554"

Optional Flags:
* -plugin-kv = '{"password":"pw here"}'
  * "password": "pw here"
  * "flip-coords": false
